Notes about Bootstrap:
  • Use Bootstrap for the layout of websites. The rest doesn't really matter, but the tools it provides for creating a layout are incredible.
  • Built for mobile devices first, works on everything thanks to "breakpoints." Give a breakpoint to an element and it limits the size of the element up to a certain size of device. Check em out if you're building a real website.
  • Containers. The go-to class for divs. Sets the maximum width of a div nicely, "contains" everything else. Throw one of these down and you're off to the races.
  • Use the grid system. It's insanely helpful for placing content in the right spot. This was a nightmare pre-Bootstrap.
  • Images used to be awful to deal with before Bootstrap. There are classes you can use on images to make them play nice in whatever box you put them in.
  • Bootstrap tables are next level. Super easy to build, and almost as easy to customize. I have used these in production-level code and they do not disappoint.
  • Bootstrap forms are almost as awesome as tables. They are really easy to use, forms are especially hard to mess with without Bootstrap because there are so many different pieces of them.
  • "Components" are pre-made... widgets... that you can use by just copy-pasting. Examples of this are the navbar, accordion, and even this list. They are super convenient and Bootstrap has an answer for just about every situation you can think of. Go find your favorite.
  • All of the stuff in this list and a lot more can be found in the docs. The docs are super easy to understand and pretty much everything in there can be mix-and-matched however you want.
